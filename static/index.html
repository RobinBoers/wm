webmention.app

Platform agnostic webmention notifications

This is a micro service that will check a given URL for links to other sites, discover if those links support webmentions and will send a webmention POST to the target.

The supported API is simplistic:

  • GET https://webmention.app/check/:url
    Perform a dry run, discovering all links and valid Webmention endpoints
  • POST https://webmention.app/check/:url
    Sends the full webmention requests
