webmention.app

This is a micro service and command line utility that will check a given URL for links to other sites, discover if those links support webmentions and will send a webmention POST to the target.

The CORS supported API is simplistic:

  • https://webmention.app/check/:url
    Perform a dry run, discovering all links and valid Webmention endpoints
  • https://webmention.app/check/:url&send=1
    Sends the full webmention requests
